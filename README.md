# Конвертер конфигурационного языка в XML
## Описание
### Инструмент командной строки для преобразования учебного конфигурационного языка в XML формат.
#### Особенности
- Полная поддержка синтаксиса учебного языка
- Все базовые операции и функции
- Генерация валидного XMLОбработка ошибок

## Установка
```
pip install lark
```
### Использование
```
python converter.py -i input.conf -o output.xml
```
### Синтаксис языка
#### Основные конструкции
```
! Комментарий
const имя = значение
(1, 2, 3)  ! Массив
table([ключ=значение])  ! Словарь
|5 + 3|  ! Выражение
Поддерживаемые операции
Арифметика: +, -, *, /
Числа: целые, вещественные, научная нотация
```
### Примеры

#### Конфигурация сервера
```
const port = 8080
const max_connections = 100
const thread_pool = |max_connections / 2|
```
#### Конфигурация игры
```
const player_health = 100
const damage = |player_health * 0.25|
const weapons = (1, 2, 3)
